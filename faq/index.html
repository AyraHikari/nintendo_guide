Apakah program ini cocok untuk komputer saya?

  • Program ini hanya cocok untuk Windows 7, 8, 8.1, dan 10 saja
  • Dan dapat berjalan pada perangkat x86 dan x64
  • Untuk windows XP sayangnya tidak bisa menggunakan program ini

Berapakah penyimpanan yang di perlukan pada program ini?

  • Untuk NES Classic Edition, dibutuhkan sekitar 600 MB ruang bebas
  • Untuk Super NES Classic Edition, dibutuhkan sekitar 1,3 GB ruang bebas

Berapakah RAM yang dibutuhkan untuk menjalankan program ini?

  • Di butuhkan minimal 256 MB RAM untuk menjalankan program ini

Apakah program ini menggunakan 'Firmware' asli dari NES Classic Edition?

  • Tidak, hanya antarmuka nya saja yang mirip
  • NES Classic Edition hanya maksimal 30 game saja, sedangkan program ini sudah terdapat 778 game NES dan 810 Super NES, Total 1588 Games (Jika beli kedua nya)

Bagaimana jika saya sudah membeli, lalu program tsb tidak dapat berjalan di perangkat saya?

  • Jika tidak dapat berjalan pada perangkat anda, anda dapat mendownload emulator nintendo/super nintendo dari website lain

Untuk folder game ada dimana?

  • Untuk NES ada di 'Nes Classic Edition/RetroArch/roms'
  • Untuk Super NES ada di 'Snes Classic PC Edition/RetroArch/roms'

Ada kah list game yang ada di paket penjualan?

  • List NES Games
  • List Super NES Games

Semua program tidak dapat berjalan di perangkat saya, mohon retur

  • Retur barang hanya akan di terima jika:
  • Flashdisk tidak tergores atau rusak (Jika bundle flashdisk)
  • CD tidak tergores atau rusak (Jika bundle CD-R/DVD-R/CD-RW/DVD-RW)
  • Garansi hanya berlaku 2 hari sejak barang di terima

Bagaimana jika Flashdisk/CD tidak bekerja?

  • Tidak ada garansi jika flashdisk/cd tidak terdeteksi, kecuali:
  • File game corrupt atau error (Sebelum pengiriman, saya tes terlebih dahulu)
  • Flashdisk minta format terus menerus (Saya jamin 100% flashdisk ini original dari pengiriman)
