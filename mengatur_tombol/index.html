Jalankan Program Nintendo

  1. Colok Flashdisk nya ke Port USB PC/Laptop anda
  2. Lalu buka program Nintendo.exe

Pengaturan

Setelah itu, klik tombol 'Pengaturan'

Mengatur Input Game

Pada awal anda akan menjalankan program yang terlihat seperti OS konsol, ikuti panduan di bawah ini

Ke bagian ikon pengaturan, dengan menekan 'arah panah kanan' pada keyboard, lalu enter

Lalu ke bagian 'Input', lalu enter

Setelah itu, ke bawah cari tulisan 'Input User 1 Binds', lalu enter

Dan cari tulisan 'User 1 Bind All', enter Dan ikuti apa yang harus di tekan pada tombol tsb

Setelah itu anda bisa keluar dari pengaturan dengan menekan tombol 'esc'

Mengatur Input Menu

Buka program Nintendo.exe jika belum, lalu klik 'Jalankan'

Tombol bawaan pengaturan adalah:

ESC = Keluar
Enter = Start
Tab = Menu Settings

Untuk mengatur menu, tekan tombol 'Tab', lalu pilih 'Controls'

Setelah itu kalian bisa mengatur kontrolnya untuk keyboard ataupun joystick
