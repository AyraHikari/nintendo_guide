Tentang

Pada software ini kita akan menjalankan NES Mini Classic dan Super NES Mini Classic pada PC Windows/Linux tanpa perlu membeli hardware official dari nintendo seharga $59.99 (30 games, NES Mini Classic) dan $79.99 (20+1 games, Super NES Classic).

Jika di hitung-hitung ke Rupiah harga tersebut cukup mahal untuk konsol modern, apalagi game nya yang cukup sedikit dibandingkan total seluruh game Nintendo lawas.

Pada software ini anda hanya perlu membeli game ini dengan harga 90% lebih murah daripada konsol aslinya, hanya cukup membeli dua atau lebih joystick, software original, anda dapat menjalankan simulasi konsol modern ini!

Di software ini terdapat total 1599 Games (778 NES, 811 SNES) yang dapat di install di Komputer anda.

Preview

Kelebihan

  • Full games dan hampir semua game ada (tanpa game hacked atau unofficial)
  • Mempunyai Save States lebih dari satu (Konsol original hanya mempunyai suspend save max 3)
  • Bisa menyimpan save seperti konsol original
  • UI dan Musik sama seperti konsol original (disarankan membeli joystick original dengan USB agar suasana konsol makin terasa)
  • Bebas menggunakan joystick apapun (dengan syarat harus konfigurasi joystik)
  • Dapat di jalankan pada NTFS dan PAL (Di sarankan terhubung ke TV agar suasana konsol makin terasa)
  • Terdapat Box Art untuk semua games (Sama seperti konsol original)
  • Filter favorit dan semua games agar tidak sulit mencari game favorit
  • Dapat bermain bersama teman dengan max 4 orang (tergantung game nya max berapa orang)

Kekurangan

  • Tidak ada suspend save, bisa di ganti menggunakan save state
  • Tidak ada pencarian dengan teks, konsol original pun tidak ada
  • Harus konfigurasi manual untuk penggunaan joystick (Cek di halaman konfigurasi joystik)
  • Sayangnya hanya support Windows 7, 8, 8.1 dan 10 saja, untuk linux akan hadir nanti
